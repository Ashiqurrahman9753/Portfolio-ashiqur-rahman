# Portfolio - Ashiqur Rahman

A modern, responsive portfolio website showcasing my journey as a Software Engineer with expertise in Network Systems and Full-Stack Development.

**Live Demo:** [portfolio-ashiqur-rahman.vercel.app](https://portfolio-ashiqur-rahman.vercel.app)

---

## Tech Stack

- **React 19** - Modern UI library
- **Vite** - Lightning-fast build tool
- **Tailwind CSS** - Utility-first styling
- **Framer Motion** - Smooth animations
- **Lucide React** - Beautiful icons

---

## Features

- Typewriter animation effect in navbar
- Fully responsive design (mobile, tablet, desktop)
- Smooth scroll navigation
- Interactive project showcases
- Downloadable CV/Resume
- Contact form integration
- Custom background styling

---

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
